# AI-Powered Spam Detection Platform

## Overview

This project is a scalable backend system for detecting spam phone numbers using:

- Django & DRF
- Redis & Celery
- Machine Learning (NLP)
- Docker & CI/CD

## Features

- Phone number lookup
- Spam detection
- Async processing
- ML-based scoring
- Scalable architecture

## Tech Stack

- Python, Django, DRF
- PostgreSQL
- Redis, Celery
- Docker
- GitHub Actions
